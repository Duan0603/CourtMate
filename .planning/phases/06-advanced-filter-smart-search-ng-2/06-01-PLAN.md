# 06-01: Backend Search & Filtering

<objective>
Enhance the backend to support searching tournaments by title/organizer and filtering by various criteria.
</objective>

<tasks>
1. Update `packages/shared/src/index.ts` to include a `TournamentFilterDto` interface (keyword, sport, city, minFee, maxFee, status).
2. Modify `apps/backend/src/modules/tournaments/tournaments.controller.ts` to accept Query parameters matching the new filter fields in `findAll`.
3. Update `apps/backend/src/modules/tournaments/tournaments.service.ts`:
   - Enhance the `findAll` method to build a dynamic MongoDB query based on the provided filters.
   - Use `$regex` (case-insensitive) for searching `keyword` across `title` and `organizer.name`.
   - Handle filtering by `sport`, `city`, and `status`.
   - Note: Categories fee filtering might require an `$elemMatch` query on the `categories` array.
</tasks>
